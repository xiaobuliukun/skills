# Skills 仓库

这是一个 **AI Agent Skills** 集合仓库，包含：

- 我收录的实用技能（精选、整理）
- 我自己编写的技能（持续迭代）

目标是沉淀一套可复用、可组合的能力模块，让日常开发和内容生产更高效。

## 仓库特点

- 主题覆盖广：前端、测试、文档、演示文稿、MCP、设计与创意等
- 每个技能独立维护：通常以目录形式存在，核心说明在 `SKILL.md`
- 可扩展：可以持续新增技能目录并补充参考文档

## 使用方式

1. 进入目标技能目录
2. 阅读该目录下的 `SKILL.md`
3. 按文档中的输入/输出约定执行
4. 如有 `reference`、`examples`、`themes` 等子目录，可作为补充资料

## 目录约定（建议）

```text
<skill-name>/
  ├── SKILL.md          # 技能主说明（必备）
  ├── reference/        # 参考资料（可选）
  ├── examples/         # 示例（可选）
  └── assets/           # 资源文件（可选）
```
